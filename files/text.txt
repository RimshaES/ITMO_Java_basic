В языке программирования Java класс — это основная строительная единица ООП. Класс представляет
собой шаблон или модель, по которой создаются объекты. Он определяет состояние (переменные) и
поведение (методы) объектов, которые создаются на основе этого класса.
Классы в Java используются для описания объектов реального мира и абстрактных концепций.
Они позволяют определить структуру данных и функционал, связанный с этой структурой данных.
